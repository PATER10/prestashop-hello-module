# PrestaShop Hello Module

Simple custom PrestaShop module created for learning purposes.

## Features
- Displays greeting message on homepage
- Detects logged user
- Changes color depending on login state
- Uses PrestaShop hooks
- Modular approach

## Technologies
- PHP
- PrestaShop
- Smarty (.tpl)
- CSS

## Installation
1. Copy module to `/modules`
2. Install module in PrestaShop admin panel
3. Hook automatically displays on homepage
